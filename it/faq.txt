  <li><b> Che cosa è Notionary ? </b>
     <p>
     Notionary è una raccolta di prove corte che aiutano a imparare qualsiasi soggetto per mezzo
     di Domande e Risposte (p.e. Capitali Europee), o Immagini (p.e. Bandiere Africane),
     o Suoni (p.e. Inni Famosi) o una combinazione (p.e. Semiografia Musicale).

  <li><b> Come posso creare una Notion ? </b>
      <p>
      Le Notions possono essere creati manualmente digitando coppie ( Domanda -->> Risposta ) nello editor fornito da Notionary.<br/>
      In alternativa, Notions possono essere caricati nel sistema se sono in in forma di un file di coppie ( Domanda -->> Risposta ) separati da " -->> ".<br/>
      Basta usare il vostro editor preferito per creare un file di tipo ".txt" o ".csv " simile all'esempio qui : <br/>
      <span class='broso'>Domanda 1</span> &nbsp;-->>&nbsp; <span class='bgrun'>Risposta 1</span><br/>
      <span class='broso'>Domanda 2</span> &nbsp;-->>&nbsp; <span class='bgrun'>Risposta 2</span><br/>
      <span class='broso'>Domanda 3</span> &nbsp;-->>&nbsp; <span class='bgrun'>Risposta 3</span><br/>
      <span class='broso'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;...</span> -->> <span class='bgrun'>...</span><br/>
      <span class='broso'>Domanda x</span> &nbsp;-->>&nbsp; <span class='bgrun'>Risposta x</span><br/>
      Dopo aver creato una Notion manualmente o tramite il carico, è possibile modificare la Notion e
      aggiungere Immagini e Suoni per ogni Domanda. Queste Immagini e Suoni dovrebbero aiutare a formare
      associazioni di Risposta senza menzionare né mostrare la Risposta. Se credi che il materiale
      si può imparare più facilmente attraverso un video o mediante la lettura di un documento,
      è comunque possibile utilizzare Notionary come una piattaforma di test e collegare il vostro video
      ( YouTube ) o allegare il documento ( PDF ) come materiale preliminare.
      </p>
