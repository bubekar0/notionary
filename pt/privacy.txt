<h4> Política de Privacidade </h4>
<p>Esta oferta é feita por Notionary ("notionary" ou "nós"), Notionary.com 01187 Dresden - Alemanha. </p>
<p>Manipulação confiável e competente dos dados pessoais dos nossos usuários é importante para nós. Se nós coletamos, transformar
   ou uso ("use") Os dados pessoais como parte de nossa oferta, isso deve ocorrer dentro do contexto das leis de protecção de dados.</p>
<p>A seguinte política de privacidade, gostaríamos de descrever de forma transparente e compreensível, que os dados de usuários do nosso serviço
    notionary são armazenados e para que fim é necessário tal utilização dos dados em cada caso. </p>
<h3> Que dados serão utilizados pela notionary </h3>
<ol>
   <li><b> Dados do cliente </b></li>
   Se você utilizar o nosso website, sem registrar, como assistir ou ler, não usar qualquer de suas informações pessoais. Ao registrar-se para
   utilizar os nossos serviços, então usamos as informações pessoais que você fornecer ativamente. Este é, em especial, as informações
   necessário para a execução do contrato que você fornecer ao se registrar. Estes incluem, por exemplo, seu nome, data de nascimento,
   endereço, endereço de e-mail, se necessário, outras informações fornecidas por você durante o processo de registro e dados
   necessário para o faturamento. Se você utilizar os serviços no contexto de livre acesso a funcionalidade limitada, os dados coletados durante o
   processo de registro, opcionalmente, podem ser limitadas a seu nome, endereço de e -mail e instituição de ensino.

   <li><b> Informações publicadas pelo usuário </b></li>
   Além disso, as informações armazenadas notionary o usuário fornecido como parte do nosso serviço. Estes incluem o nome do usuário,
   e, possivelmente, quaisquer comentários postados pelos usuários em nosso site. Lembramos os nossos utilizadores que a decisão quanto ao que
   eles compartilham sobre si está totalmente em suas mãos.

   <li><b> Dados de nível aprendizagem </b></li>
   No contexto da utilização de nossos serviços de aprendizagem serão coletadas e utilizadas ("Dados de nível de aprendizagem") por notionary
   a fim de proporcionar aos participantes com o melhor uso de serviços de aprendizagem . A informação do nível de aprendizagem inclui, por exemplo,
   o número de visitas para cada tarefa de aprendizagem, o estado atual do nível de aprendizagem, o tempo necessário para cada tarefa de aprendizagem
   e os resultados de cada tarefa de aprendizagem e assunto. Estes dados fornecem aos usuários uma melhor orientação quanto à sua
   progresso na aprendizagem e permitir notionary sugerir novos conteúdos para os usuários com base em seus níveis de habilidade.
   Os dados do nível de aprendizagem são visíveis apenas para o respectivo usuário ou contraparte do notionary contrato. Em nenhum caso a transmitir
   aprender dados do nível a terceiros sem o seu consentimento.

   <li><b> Uso de Dados </b></li>
   Na medida do necessário para oferecer aos nossos serviços no âmbito do acesso ao nosso serviço, podemos extrair dados de utilização, tais
   como o nome do usuário, data e hora de cada acesso. No contexto das disposições legais, podemos avaliar os dados para a publicidade,
   pesquisa de mercado e para melhorar os nossos serviços através de perfis usando pseudônimos. Você pode contestar este uso em qualquer
   tempo sem dar razões.
